# GeoServer Configuration

Ce répertoire contient la configuration GeoServer pour les cartes interactives.

## Accès
- **URL**: http://localhost:8080/geoserver
- **Admin**: admin / aquawatch123

## Configuration PostGIS
Connecter à la base geo_db (localhost:5436) pour servir les couches cartographiques.

## Couches à configurer
- Zones de surveillance
- Points capteurs
- Prédictions géolocalisées
- Alertes par zone
